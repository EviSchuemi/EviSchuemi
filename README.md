# EviSchuemi

## Research focus:
Sleep electrophysiology, Neuromodulation (auditory stimulation, transcranial temporal interference stimulation (tTIS)), memory consolidation
## Short Description:
My PhD investigates the neural mechanisms governing memory consolidation during sleep. Our project combines auditory stimulation with temporal interference stimulation to target and modulate the neural mechanisms responsible for memory processing.
